NAME

gcloud beta firestore fields - manage field metadata for Cloud Firestore

SYNOPSIS

`gcloud beta firestore fields` `  GROUP  ` \[ `  GCLOUD_WIDE_FLAG …  ` \]

DESCRIPTION

`(BETA)` Manage field metadata for Cloud Firestore.

GCLOUD WIDE FLAGS

These flags are available to all commands: `  --help  ` .

Run ` $ gcloud help  ` for details.

GROUPS

`  GROUP  ` is one of the following:

  - `  ttls  `  
    `(BETA)` Manage Time-to-live metadata for Cloud Firestore.

NOTES

This command is currently in beta and might change without notice. These variants are also available:

    gcloud firestore fields

    gcloud alpha firestore fields
