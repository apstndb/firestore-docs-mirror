# Implement an AI-driven chatbot

Use Firebase Extensions to add an AI-driven chatbot to your app. You can use chatbots to improve customer service, market a new feature or service, provide sales quotes, or any number of use cases.

## Implement a PaLM-driven chatbot

Use this extension to easily deploy a chatbot using the PaLM API, stored and managed by Firestore in Native Mode. [Learn more](https://extensions.dev/extensions/googlecloud/firestore-palm-chatbot) .
