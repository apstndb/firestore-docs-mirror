# Enrich handling of text

Use Firestore in Native Mode and Firebase Extensions to enrich your ability to work with text assets.

## Summarize text

This extension allows you to summarize a field in a Firestore in Native Mode document using the PaLM API. [Learn more](https://extensions.dev/extensions/googlecloud/firestore-palm-summarize-text) .

## Detect toxic speech

Use this extension to get toxicity scores for user-submitted text, like comments on social posts, written to a Firestore in Native Mode collection. [Learn more](https://extensions.dev/extensions/jigsaw/firestore-perspective-toxicity) .

## Translate text

Translate strings written to a Firestore in Native Mode collection into multiple languages using the Cloud Translation API. [Learn more](https://extensions.dev/extensions/firebase/firestore-translate-text) .

## Transcribe audio

This extension transcribes audio to text in a Firestore in Native Mode collection. [Learn more](https://extensions.dev/extensions/googlecloud/speech-to-text) .
