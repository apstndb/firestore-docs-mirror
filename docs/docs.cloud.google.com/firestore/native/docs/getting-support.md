# Getting support

## Get a Google support package

Google Cloud offers different support packages to meet different needs, such as 24/7 coverage, phone support, and access to a technical support manager. For more information, see [Cloud Customer Care](/support) .

## Get support from the community

### Ask a question on Stack Overflow

Ask a question about Firestore on [Stack Overflow](http://stackoverflow.com/questions/tagged/google-cloud-firestore) . Use the tag `  google-cloud-firestore  ` for questions about Firestore. This tag not only receives responses from the Stack Overflow community, but also from Google engineers, who monitor the tag and offer unofficial support.

### Discuss Firestore

You can visit the Google Cloud [Slack community](https://googlecloud-community.slack.com/) to discuss Firestore and other Google Cloud products. If you haven't already joined, [use this form to sign up](https://join.slack.com/t/googlecloud-community/shared_invite/zt-3icuwfdvq-eq_58LqmOoCNc16mpZcaNA) . For Firestore, join the [`  #firestore  `](https://googlecloud-community.slack.com/messages/C7JGTJ0DQ/) channel.
