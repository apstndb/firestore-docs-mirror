## Get a Google support package

Google Cloud offers different support packages to meet different needs, such as 24/7 coverage, phone support, and access to a technical support manager. For more information, see [Cloud Customer Care](/support) .

## Get support from the community

### Ask a question on Stack Overflow

Ask a question about Datastore mode on [Stack Overflow](http://stackoverflow.com/questions/tagged/google-cloud-datastore) . Use the tag `  google-cloud-datastore  ` for questions about Datastore mode. This tag not only receives responses from the Stack Overflow community, but also from Google engineers, who monitor the tag and offer unofficial support. Please review the [most-asked questions](http://stackoverflow.com/questions/tagged/google-cloud-datastore?sort=frequent) before posting a new one.

### Discuss Datastore mode

Join the [gcd-discuss](https://groups.google.com/forum/#!forum/gcd-discuss) Google group to discuss Datastore mode and receive Datastore mode announcements and updates.

You can also take the following action to engage with the Datastore mode community:

  - Visit the Google Cloud [Slack community](https://googlecloud-community.slack.com/) to discuss Datastore mode and other Google Cloud products. If you haven't already joined, [use this form to sign up](https://join.slack.com/t/googlecloud-community/shared_invite/zt-3icuwfdvq-eq_58LqmOoCNc16mpZcaNA) . For Datastore mode, join the [`  #datastore  `](https://googlecloud-community.slack.com/messages/C0HSDJBSR/) channel.

## File bugs or feature requests

From the Datastore mode documentation click "Send feedback" near the top right of the page. This will open a feedback form. Your comments will be reviewed by the Datastore mode team.
